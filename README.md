# **_반응형 웹 OTT TAING 구현하기_**

## 목차 및 기간

- 목차

  1. 메인 페이지
  2. 검색창
  3. 랜딩 페이지
  4. 로그인 페이지
  5. 회원가입 페이지
  6. 아이디/비밀번호 찾기 페이지

- 기간 : 2023 / 06 / 23 ~ 2023 / 06 / 28

## 프로젝트 진행 시 준수사항

- 적절한 헤딩 사용하기, 시맨틱 마크업에 신경쓰기
- 문법 오류 없도록 구현하기
- 페이지 단위로 구현하기
- 네이밍 시 BEM 패턴 사용하기
- 모두가 편하게 사용할 수 있는 웹 구현하기
