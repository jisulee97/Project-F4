# OTT TAING 구현하기

## **목표 및 기간**

- 목표 : 메인페이지와 로그인, 아이디/비밀번호 찾기, 회원가입 페이지 구현하기

- 기간 : 2023 / 06 / 23 ~ 2023 / 06 / 28

## **프로젝트 진행 시 준수사항**

- 적절한 헤딩 사용하기, 시맨틱 마크업에 신경쓰기
- 문법 오류 없도록 구현하기
- 페이지 단위로 구현하기
- 네이밍 시 BEM 패턴 사용하기
- 모두가 편하게 사용할 수 있는 웹 구현하기
